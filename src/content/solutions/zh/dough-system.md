## 系统组成

自动加面系统包含叠面输送模块、面块整形机构、缓存输送模块、爬坡输送模块、输送小车导轨综合线、输送小车、控制系统和监测系统等部分。

### 叠面输送模块
面带从压面输送带出来后，经爬坡输送带输送至叠面模块，完成面带的初步形态，后经过整形辊筒，对面带进行整形处理。

### 缓存输送模块
用于衔接面皮输送工序与输送小车取料工序，起到工序缓冲、物料储备的作用。当供面小车抵达取料位置时，可快速完成面块补给。

### 输送小车导轨综合线
加馅轨道的横向支撑轴向外延伸后增设独立支撑立腿，将加面轨道直接架设于该延伸轴与新增立腿构成的支撑结构上。

### 材质标准
- 全线设备框架、辊筒、轴等采用食品级304不锈钢材质
- 输送带为白色PU专用材质
- 符合食品安全标准

### ZGM-V3.0 复合柔性供料调度系统

博瑞卡自研ZGM-V3.0调度软件，与ZGL-V3.0加馅调度系统形成互补：
- 实时抓取监测数据，动态匹配供料运投与生产需求
- 柔性参数设置，一键启动/一键停止
- 数据统计：记录整条线及每台成型机的产能数据
- 支持手动模式切换
- 传感器采集无延迟，输送小车自动更新数据

### 输送小车（核心）

- 悬挂式结构，304不锈钢，全钣金封闭
- 输送带承重≥100kg，白色PU食品级材质
- 大直径不锈钢包塑车轮，伺服驱动控制
- 双向传感器+机械硬限位双重安全保障
- 多点传感器精准把控面块位置、间距和数量

### 安全与卫生设计

- 每台成型机工位三色灯可视化预警（绿/黄/红闪）
- 防撞网板保护下方检修人员安全
- 自带导轨除尘系统，降低卫生清扫劳动强度
- 无复杂多组皮带，彻底消除积料死角
- 伺服小车无需齿轮油，杜绝油污风险

## 产品优势（对比传统皮带输送）

| 对比项 | 传统皮带输送 | 博瑞卡ZGM-V3.0系统 |
|--------|------------|-------------------|
| 空间占用 | 大，多节点 | 紧凑布局，标准化结构 |
| 故障风险 | 皮带节点多，故障点高 | 模块化简洁设计，故障点少 |
| 卫生 | 积料死角多，难清理 | 无复杂多组皮带，卫生无死角 |
| 油污风险 | 减速电机齿轮油泄漏 | 伺服无齿轮油，零泄漏 |
| 控制精度 | 准确性/同步性差 | 伺服闭环控制，S型加减速零冲击 |
| 噪音 | 噪声大 | 低噪音运行 |
| 能耗 | 多电机空载损耗大 | 单伺服+低阻尼导轨，高效节能 |
| 面块质量 | 易堆叠/挤压/偏移 | 零冲击输送，面筋组织不受破坏 |
| 扩展性 | 改造工期长 | 预留接口，方便扩展 |
| 移机 | 拆卸损耗大 | 模块化100%重复利用 |
